最大平均值问题(USACO2003 March Open)

[题意简述]

读入一列正数，a1,a2,…,aN，以及一个数F。定义，

求Max{ave(a,b),1≤a,b≤N，且a≤b-F+1}，即求一段长度大于等于F且平均值最大的子串。

范围：F≤N≤105。

[分析]

枚举肯定不行。但可以使用图形化。

令前缀和

如果将S函数绘在平面直角坐标系内，这就是过点和点直线的斜率！

于是问题转化为：平面上已知N+1个点，，0≤i≤N，求横向距离大于等于F的任意两点连线的最大斜率。

数据如下时

即求之中的最大斜率

显然要维护一个凸包

接下来请移步《浅谈数形结合思想在信息学竞赛中的应用》周源

斜率优化

下面正式讲斜率优化。

对于一类Dp方程式：f[i]=min(f[j]-a[i]∗b[j]+c[j]+d[i]) a[x],b[x],c[x],d[x]都为x的函数，且a单增。

在 i 时对于最优决策 j : f[i]  = f[j] -a[i] * b[j]+ c[j] +d[i]
                     (f[i]-d[i])= -a[i] * b[j]+ (c[j]+f[j])             
                         b    =      -k    *  x  +  y   (y=kx+b)

对于任意j可以表示为图像上一点 J ( b[j],c[j]+f[j] )，目的是求出f[i],因为f[i]的值等于y=kx+b直线的截距。

添加一个点时，我们就确定了斜率k = a[i]，x、y在已有的点中选择。

问题转化为：对于一条斜率k = a[i] 的直线（为什么将它作为斜率?），使其至少过一点J(即上文那个任意J点,j<i)，使得该直线的截距最小(就是与y轴焦点的纵坐标值)。那么考虑哪些J点有望成为答案？

因为a[i]单调递增，即斜率逐渐变大。所以可以维护一个下凸包。逐渐去除尾部

而加入的点用上面的方式去除不可能成为最优解的解。

在代码实现“去除尾巴”和“清理头部”中，都是以判断斜率为标准,所以是叫斜率优化

回归数学

还记得决策单调性和算斜率的一大堆推导吗？

决策单调性就是证明了d[i]对每次决策没有影响（既然在求i时，j比k要优，那么之后，即在t(t>i)时，j是否也比k要优）。似乎是在证a单增。这步似乎没什么用特别是对a[i]不单增时更没用

而算斜率就是从两个J ( b[j],c[j]+f[j] )中算出斜率。

题目描述

[题目传送门](https://www.luogu.org/problemnew/show/P3195)

P教授要去看奥运，但是他舍不下他的玩具，于是他决定把所有的玩具运到北京。他使用自己的压缩器进行压缩，其可以将任意物品变成一堆，再放到一种特殊的一维容器中。P教授有编号为1⋯N 的 N 件玩具，第 i 件玩具经过压缩后变成一维长度为 C_i .为了方便整理，P教授要求在一个一维容器中的玩具编号是连续的。同时如果一个一维容器中有多个玩具，那么两件玩具之间要加入一个单位长度的填充物，形式地说如果将第 ii 件玩具到第 j 个玩具放到一个容器中，那么容器的长度将为$ x=j-i+\sum\limits_{k=i}^{j}C_k$制作容器的费用与容器的长度有关，根据教授研究，如果容器长度为 x ,其制作费用为 $(X-L)^2$ .其中 L 是一个常量。P教授不关心容器的数目，他可以制作出任意长度的容器，甚至超过 L 。但他希望费用最小.

发现 $ f[i]=min^{j=0..i-1}(f[j]+(i-j-1+s[i]-s[j]-L )^2)$可以使用斜率优化
