问题 $L1^1$: 棋盘DoubleColorChessboard

​ $L^n定义为我的题目编号$

时间限制: 1 Sec 内存限制: 256 MB

题目描述

玩个游戏:给出一个 n 行 n 列的棋盘,里面有 n*n 个方格,其中每个格子上有颜色,总共两种颜色 0 和 1。 Bob 要遵循规则去玩:一张卡要正好覆盖两个相邻且同色的正常格子。我们的任务是帮助 Bob 知道棋盘在上述规则下能覆盖多少格子。

输入

第 1 行:整数 n 表示棋盘的长和宽。
第 2 行到第 n+1 行,有 n 个数,表示棋盘颜色。

输出

整数:能覆盖多少格子

样例输入

样例输出

提示

1≤n≤100
